# InstaSearch 📸

Welcome to **InstaSearch**, a sleek and modern web application that brings you stunning images directly from Unsplash! 🌟 Dive into a world of creativity and discover visuals that match your imagination. This project utilizes the Unsplash API to fetch and display high-quality images based on your search queries.

---

## 🚀 Features

1. **Dynamic Image Search** 🖼️
   - Enter a keyword in the search bar to fetch 10 stunning images related to your query.
   
2. **Responsive Design** 📱
   - Tailwind CSS ensures the application looks amazing on all devices.

3. **"Generate More" Button** 🔄
   - Fetch additional images seamlessly for an endless browsing experience.

4. **Error Handling** ❗
   - Displays an error message if the search fails or returns no results.

5. **Simple and Clean UI** 🎨
   - User-friendly interface to make searching images effortless.

---

## 🌈 How It Works

1. User enters a search term in the input field.
2. The app sends a request to the Unsplash API using the search query.
3. Displays a grid of 10 beautiful images fetched from Unsplash.
4. Users can load more images by clicking on the "Generate More" button.

---

## 🛠️ Technologies Used

- **Frontend:** HTML5, CSS3 (Tailwind CSS), JavaScript (Vanilla JS)
- **API Integration:** [Unsplash API](https://unsplash.com/developers)
- **Version Control:** Git, GitHub

## 🔧 Setup and Installation

Follow these steps to get InstaSearch running on your local machine:

1. Clone the repository:
   ```bash
   git clone https://github.com/Kethanvr/InstaSearch.git
   ```

2. Navigate to the project directory:
   ```bash
   cd InstaSearch
   ```

3. Open the `index.html` file in your favorite browser:
   ```bash
   open index.html
   ```

That’s it! 🎉 Start searching for amazing images!
